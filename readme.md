# <center>Scout Bot</center>

### Environment Variables

Copy [.env.example](./.env.example) file to a new `.env` file and then fill up the required info.

### Install

```bash
pnpm install
```

### Run in dev mode

```bash
pnpm dev
```

### Run in production

```bash
pnpm start
```

### Deploy Commands to Discord

```bash
NODE_ENV=production pnpm deploy:commands
```

### Code Lint

```bash
pnpm lint
```

### Code Format

```bash
pnpm format
```
